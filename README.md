# 📦 Gestão de Estoque

Sistema simples de gerenciamento de produtos com API em Java (Spring Boot) e interface web.

---

## 🚀 Funcionalidades

- Listar produtos  
- Adicionar produto  
- Editar produto  
- Remover produto  

---

## 🛠️ Tecnologias

- Java + Spring Boot  
- SQL  
- HTML, CSS, JavaScript  

---

## 🔌 Endpoints

- `GET /produtos`  
- `POST /produtos`  
- `PUT /produtos/{id}`  
- `DELETE /produtos/{id}`  

---

## 💡 Como rodar

- Backend: rodar o Spring Boot  
- Frontend: abrir com Live Server
